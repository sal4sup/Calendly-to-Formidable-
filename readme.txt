Sync Calendly bookings into Formidable Forms using a Calendly Personal Access Token.

== Description ==
Calendly to Formidable Bridge syncs Calendly invitee events into Formidable Forms Form ID 4.

Features:
- Personal Access Token integration
- Webhook endpoint
- Webhook management actions
- Duplicate prevention with plugin-owned mappings table
- Diagnostics and recent bookings
- Manual payload test tool

== Installation ==
1. Upload the `calendly-to-formidable-bridge` plugin folder to `/wp-content/plugins/`.
2. Activate the plugin from WordPress admin.
3. Go to Settings -> Calendly to Formidable.
4. Save your Calendly Personal Access Token.
5. Click Test Connection.
6. Click Create Webhook.
7. Verify Recent Bookings.
8. Create a Calendly booking and confirm Formidable entry in Form ID 4.
